# AWave
## WSS Server (Go)

```bash
cd ~/ws_node/awave/src/fileServer/src && go run cmdServer 127.0.0.1:4000
```

## Controller Client (React Native)

```bash
cd ~/ws_node/awave/src/mobile/awave && yarn web
```

## Music Client (NextJs)

```bash
cd ~/ws_node/awave && npm run dev
```
